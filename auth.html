Sign in with Google

CrowdCare is Solana-first. After you sign in, we derive a demo Solana public key from your Google account id so you have a consistent address to receive funds in this prototype.

Set up Google Sign-In. Copy config.example.js to config.js (or edit config.js) and paste your Web client ID from Google Cloud Console. Add your GitHub Pages origin and/or http://localhost:PORT under Authorized JavaScript origins. Google Sign-In does not work from a file:// URL—use a local server or your live site.
Demo only. This key is produced in the browser from a hash of your Google id plus a fixed app string. It is not a custodial wallet, a standard seed phrase, or production key management. Before real money, plan for a proper wallet (e.g. embedded-wallet provider or user-owned wallet) and legal review.

After a successful sign-in you’ll return home with your Solana address stored in this browser only.
