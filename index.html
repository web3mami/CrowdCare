Crowdfunding on Solana. Sign in with Google to get a demo receive address for this browser, then explore and start campaigns.

Set up Google Sign-In. Edit config.js and paste your Web client ID from Google Cloud Console. Add your site origin under Authorized JavaScript origins. Sign-in does not work from file://—use GitHub Pages or a local server.
Demo only. Your Solana public key here is derived in the browser for prototyping—not a production wallet. Review before real funds.
